# Change Log

All notable changes to the "Polaris Search" extension will be documented in this file.

## [0.1.0] - 2026-01-22

### Initial Release

- **Find Files**: Fuzzy search through workspace files with `Cmd+Alt+F` (Mac) or `Ctrl+Alt+F` (Windows/Linux)
- **Find in Files**: Content search using ripgrep with `Cmd+Alt+P` (Mac) or `Ctrl+Alt+P` (Windows/Linux)
- **Find in Open Files**: Search within currently open editor tabs with `Cmd+Alt+O` (Mac) or `Ctrl+Alt+O` (Windows/Linux)
- **Replace Support**: Replace text across files with preserve-case logic
- **Syntax Highlighting**: Beautiful code preview powered by Shiki with support for 15+ themes
- **Modern UI**: Clean, responsive webview interface
- **Configurable**: Customize themes, preview lines, search delay, and more

### Features

- Fuzzy file finding with file icons
- Fast content search powered by ripgrep
- Real-time search as you type
- Context preview with syntax highlighting
- Line numbers in preview pane
- Multiple theme support (Tokyo Night, Dracula, GitHub, Nord, One Dark Pro, and more)
- Configurable search delay and preview lines
